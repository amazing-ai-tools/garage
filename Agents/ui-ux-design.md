# UI/UX Design — garage

Use this skill for screen design, layout, copy, and interaction work.

## Product State
garage is still a scaffold. Design work should create the actual usable first screen once the user picks a product direction.

## Standards
- Mobile-first layout.
- Accessible focus states and semantic controls.
- No oversized marketing-only shell unless the user asks for a landing page.
- Use lucide-react icons for common actions.
- Keep text short and domain-specific.
- Include empty/error/loading states for any async behavior.

## Visual Direction
- Clean operational utility.
- Avoid generic gradient-heavy template aesthetics.
- Let the chosen garage domain drive colors, imagery, and information density.
